# Product Roadmap

This roadmap describes planned improvements for the FIND AI demo product.

## Near Term

- Add markdown file upload from the Streamlit sidebar.
- Show ingestion status without blocking the full interface.
- Add source filtering by department or document type.
- Improve empty-state guidance when no index exists.

## Medium Term

- Add evaluation questions for retrieval and answer quality.
- Add support for PDF and text files.
- Support multiple named knowledge bases.
- Add conversation export for debugging and demos.

## Later

- Add role-based document access.
- Add document freshness warnings.
- Add automatic duplicate detection.
- Add admin analytics for common unanswered questions.

## Roadmap Rule

Privacy, citation quality, and simple user experience should remain more
important than adding many model features.
