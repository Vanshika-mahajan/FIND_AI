# Remote Work Policy

FIND AI supports remote-first work. Employees may work from home or another
stable work location if they can maintain communication, security, and delivery
expectations.

## Availability

Employees should be reachable during core collaboration hours from 10:00 AM to
3:00 PM local team time, Monday through Friday.

## Meetings

Team meetings should include a written agenda. Decisions made in meetings should
be summarized in a shared document so they can be searched later.

## Security

Employees must use approved devices, keep systems updated, and avoid storing
customer documents in personal cloud accounts.

## Travel

Employees working while traveling should notify their manager if time zone
changes will affect availability or meetings.
